## 介绍

在这个项目中，你将结合 JavaScript、HTML 和 CSS 来创建一个交互式网站，用户可以在其中设计自己的超级英雄角色！

<p style="border-left: solid; border-width:10px; border-color: #0faeb0; background-color: aliceblue; padding: 10px;">
<span style="color: #0faeb0">**JavaScript**</span>是 Web 编程语言，用于使网站动态化和交互化。 它是一种非常流行的编程语言，没有它，网站就会变得静态和无聊。
</p>

你将要：

- 使用 JavaScript 更改网页上的文本
- 添加表单和输入选项，让人们创建超级英雄角色
- 使用 JavaScript 显示和隐藏网站上的元素
- 为你的网站添加浅色和深色模式选项
- 保留用户偏好
- 创建一个滚动浏览图像的轮播图
